# Sauber & Zauber — Luxury Cleaning Atelier

A cinematic, animation-heavy luxury brand site built on Next.js 15, TailwindCSS, GSAP, Framer Motion, Lenis, and React Three Fiber.

## Stack
- **Next.js 15** (App Router, RSC + client components)
- **TailwindCSS 3** with bespoke design tokens
- **Framer Motion 11** — choreographed motion, scroll-linked transforms, magnetic UI
- **GSAP 3** — available for advanced timelines
- **Lenis** — inertia-based smooth scrolling
- **@react-three/fiber + drei + three.js** — cinematic hero scene with floating glass orbs, fog, transmission shaders, pointer parallax
- **TypeScript** strict mode

## Sections
1. Hero cinematic intro (3D scene + parallax title + interactive cursor)
2. Manifesto (blur-to-focus storytelling)
3. Services (horizontally scroll-pinned, morphing cards)
4. Before / After (drag-to-reveal slider, CSS-rendered scenes)
5. Process (animated timeline with scroll-grown gradient line)
6. Testimonials (floating, tilted glass cards)
7. Pricing (cadence toggle with layout animation)
8. FAQ (accordion with progress-fill reveal)
9. Contact (premium form, interactive chips)
10. Footer (mega-title parallax, aurora background)

## Premium interactions
- Custom magnetic cursor with variant states
- Magnetic buttons with elastic spring
- Split-text reveal (word & character)
- Scroll-linked parallax & blur-to-focus
- Mouse-follow ambient lighting
- Dynamic loading screen with progress
- Theme switching (dark / light) with CSS variable transitions
- Animated marquee, aurora gradients, noise textures

## Install
```bash
npm install
npm run dev
```
Open <http://localhost:3000>.

## Build
